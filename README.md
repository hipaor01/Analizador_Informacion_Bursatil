# Utilización
Las acciones disponibles para consultar van a ser las de las empresas: Apple, Microsoft, Alphabet, Amazon, Nvidia, JPMorgan, Goldman Sachs, Coca-Cola, McDonald's, Tesla, ExxonMobil, Johnson & Johnson, Pfizer. Por otro lado, los índices serán: S & P 500, Nasdaq, Dow Jones, Euro Stoxx, Nikkei, MSCI World. El modo de ejecución es el siguiente:
python main.py --accion [Accion] --indice [Indice]. 
Ejemplos de uso serían python main.py --accion Apple o python main.py --indice Nasdaq. Importante en el caso de nombres con el carácter & pasarlos entrecomillados, python main.py --accion "Johnson & Johnson".
